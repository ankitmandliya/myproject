# 10F10_hrms_financial_year_closing.md

# Module

HRMS → Leave Management → Financial Year Closing

---

# Objective

Implement a complete **Financial Year Closing & New Financial Year Opening** module for Leave Management.

This module finalizes one financial year and prepares the next one using the existing Leave Policy, Leave Balance, Carry Forward, Prorata Allocation, Attendance, and Holiday engines.

The module must be **idempotent**, **auditable**, and **rollback-safe**.

No leave calculation logic should be duplicated.

---

Laravel 10 only.
